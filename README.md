# Intern Attendance Tracking System (MVP)

A web application designed to track intern daily check-ins and check-outs, featuring a centralized administrative dashboard for real-time monitoring.

## 🚀 Overview

This Minimum Viable Product (MVP) fulfills the core requirements for tracking intern hours securely and transparently. It provides a simple, frictionless onboarding/login flow for interns and an analytical overview for administrators.

### Core Features
*   **Frictionless Intern Login:** Interns punch in using their Name, Email, and Phone Number.
*   **Automated Timestamping:** Captures exact `loginTime` and `logoutTime` upon action.
*   **Admin Dashboard:** A consolidated, high-level view enabling administrators to track all active and historical logs.
*   **State Persistence:** Powered by a clean, underlying Object-Oriented design structure.

---

## 🛠️ Tech Stack

*   **Backend:** Node.js, Express.js
*   **Frontend:** HTML5, Tailwind CSS (via CDN)
*   **Database:** MongoDB / Lightweight Local Memory State (for rapid MVP evaluation)
